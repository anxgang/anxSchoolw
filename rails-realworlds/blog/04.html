Blog 實戰 - 04. 控制器和視圖 ​

繼上一節我們成功建立的 model Post 並且塞入資料之後，

我們還沒有把資料取出來顯示在網頁上

該怎麼辦呢？讓我們繼續看下去

controller ​

終於又回到我們功能清單第一項

  • 前台
    • 訪客可以瀏覽文章清單(index)

現在我們該把資料從資料庫抓出來了

我們回到之前寫好的 post_controller.rb 中的 index 這個 action

把 @post = Post.all 補上去，也就是把所有文章取出來的意思

rb
# app/controller/posts_controller.rb
class PostController
  def index 
    @post = Post.all
  end
end
# app/controller/posts_controller.rb
class PostController
  def index 
    @post = Post.all
  end
end

在 rails 的 controller 中，只要宣告帶有 @ 的變數，就代表可以傳到 view 中使用

view ​

所以我們在 view 中直接在 <% %> 中寫上的 ruby 語法

並把剛剛從 controller 帶過來的變數顯示出來

html
<!-- app/view/posts/index.html.erb -->
<h1>Posts</h1>
<ui>
<% @posts.each do |post| %>
  <li><%= post.title %> / <%= post.content %></li>
<% end %>
</ui>
<!-- app/view/posts/index.html.erb -->
<h1>Posts</h1>
<ui>
<% @posts.each do |post| %>
  <li><%= post.title %> / <%= post.content %></li>
<% end %>
</ui>

這樣就完成顯示囉！
